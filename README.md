## Build Setup

``` bash
## Software required
Nodejs
Npm
Mongodb
Any Code Editor (Prefered to use VS Code)

Front
Back

# Front - install dependencies
npm install
npm start

# Run the Seeder
please be in folder back to install seeder
npm run seeder

# run all tests

npm test
```
